# Configuración hacia Estadísticas / UTNGolCoin (equipo)

Para desarrollo local del panel administrativo:

```json
"ApiEstadisticas": {
  "BaseUrl": "http://localhost:8080/demo/api/v1/"
}
```

Opcional (estado de servicios):

```json
"ApiUtnGolCoin": {
  "BaseUrl": "http://localhost:5000/api/"
}
```

No usar túneles ngrok en demos locales del equipo.
